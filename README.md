# 💰 Tax Loss Harvesting App

A React-based Tax Loss Harvesting dashboard that allows users to analyze capital gains and simulate tax-saving strategies by selecting assets.

---

## 🚀 Features

* 📊 Pre & Post Harvesting Capital Gains
* 🔄 Real-time update on asset selection
* ✅ Select / Deselect holdings
* 📉 Profit & Loss calculation (STCG & LTCG)
* 📌 "View All" toggle for holdings
* ℹ️ Tooltip & Notes section
* 🎨 Responsive UI (Tailwind CSS)

---

## 🛠 Tech Stack

* React.js
* Tailwind CSS
* Vite
* JavaScript

---

## 📦 Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/tax-harvesting-app.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm run dev
```
